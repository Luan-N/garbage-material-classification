# Data Folder Instructions

This project uses the **RealWaste Image Classification Dataset** from Kaggle.

## Dataset Source
https://www.kaggle.com/datasets/joebeachcapital/realwaste

## How to Set Up the Data

1. Go to the dataset link above.
2. Download the dataset as a ZIP file.
3. Extract the contents.
4. Place the extracted folder inside this `data/` directory.

The folder structure should look like this:
```text
data/
└── RealWaste/
    ├── Cardboard/
    ├── Food Organics/
    ├── Glass/
    ├── Metal/
    ├── Miscellaneous Trash/
    ├── Paper/
    ├── Plastic/
    ├── Textile Trash/
    └── Vegetation/
```
